C++ implementation of SHA-2:
- SHA-224
- SHA-256
- SHA-384
- SHA-512
- SHA-512/224
- SHA-512/256

## Supported Compilers

- Visual C++ 14.0 (Visual Studio 2015)
- CLang 3.7
